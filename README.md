# Claude Code Skills

我的自定义 Claude Code 技能集合。

## 技能列表

| 技能 | 描述 |
|------|------|
| `web-access` | 联网操作处理（搜索、网页抓取等） |
| `java-component-impl` | Java 组件辅助实现 |
| `notebooklm` | Google NotebookLM 查询 |
| `diffs` | 生成可分享的 diff |
| `docx` | Word 文档处理 |
| `frontend-design` | 前端界面设计 |
| `pdf` | PDF 文件处理 |
| `skill-creator` | 创建和管理技能 |

## 安装

1. 克隆此仓库：
```bash
git clone https://github.com/richal/my-skills.git
```

2. 将技能目录链接到 `~/.cc-switch/skills/` 或使用时的相对路径。

## 使用说明

每个技能目录下包含 `SKILL.md` 或 `README.md` 详细说明其用法。
